📋 Step-by-Step: Build It in 15 Minutes

Minute 1–3: Build in Google AI Studio

  1. Go to aistudio.google.com
  2. Start a new Applet
  3. Design your landing page, checkout, or digital product
  4. Click Deploy → Push to GitHub

Minute 4–6: Push to GitHub

  1. Create a new repo: your-product-name
  2. Make it Public
  3. Add a vercel.json file if needed:
{ "buildCommand": "npm run build", "outputDirectory": "dist", "framework": "vite" }

Minute 7–10: Deploy to Vercel

  1. Go to vercel.com → Add New → Project
  2. Import your GitHub repo
  3. Vercel auto-detects the framework
  4. Click Deploy

Your site is live at: yourproject.vercel.app

Minute 11–13: Connect Your Domain

  1. In Vercel → Project → Settings → Domains
  2. Add your domain (e.g. yourdomain.com)
  3. Vercel gives you DNS records to add in GoDaddy:
TypeNameValue
A@216.198.79.1
CNAMEwwwcname.vercel-dns.com

Minute 14–15: Set Up Payments

  1. Go to gumroad.com — free to join
  2. Create a product: "$9 PDF Guide", "$19 Template Pack"
  3. Copy your Gumroad product link
  4. Add the link to your Vercel site

Gumroad handles: credit cards, PDF delivery, refunds, receipts — automatically.

❓ FAQ

"Do I need to know how to code?"

No. Google AI Studio handles the building. You describe what you want, it builds it. We've built entire sites without writing a single line of code.

"What if Google AI Studio doesn't export my code?"

You can download the built files and push them to GitHub manually. Most AI Studio applets export cleanly to a standard Vite/React project.

"Can I use a different payment processor?"

Yes — Gumroad is the easiest (zero monthly fee, pays on sale only). Alternatives: LemonSqueezy, Stripe (requires more setup).

"What's the catch with free tiers?"

Each tool has limits, but for a new product they're more than enough. Vercel free tier handles 100K visitors/month. Supabase free tier handles 500MB database. Gumroad takes 10% only when you make a sale.

"Can I sell a recurring subscription?"

Gumroad doesn't do recurring, but LemonSqueezy and Stripe do. For $9/mo subscriptions, consider LemonSqueezy — similar free tier.
